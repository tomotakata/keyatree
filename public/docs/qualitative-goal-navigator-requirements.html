定性目標設定ナビゲーター機能
要件定義書

「誰でも簡単にできる目標設定シート（PDF）」を唯一の基盤とした、定性目標設定ナビゲーター機能をシステム化するための要件を整理した文書です。

対象機能定性目標設定ナビゲーター
対象システムKeyaTree
作成日2025年6月

目的

PDF準拠のステージ・グレード・カテゴリ・コンピテンシーを基盤とし、定性目標を迷わず設定できる伴走型ウィザードを構築する。最終的には、本人の言葉による目標と行動計画を Word提出前提のレポートとして出力できる状態を実現する。

1

機能概要

対象ユーザー

  • 一般社員（定性目標設定者）
  • 上長（確認・差戻し・承認）
  • 人事管理者（全体管理・マスター管理）

基本方針

  • PDFを唯一の基盤とする
  • 選択項目の言い換え・要約は禁止
  • 質問は1回に1つだけ
  • 必ず行動3つまで落とし込む
2

進行フロー要件

INIT
初期ヒアリング

名前、部署を順に1問ずつ取得する。完了までPRE-STEPへ進めない。

PRE-A
ステージ選択

PDFに定義されたステージから選択させる。言い換え・要約は禁止。

PRE-B
グレード選択

選択したステージに紐づくグレード候補を提示し、確定する。

PRE-C
定性目標カテゴリ選択

PDF準拠のカテゴリをそのまま提示し、選択させる。

PRE-D
コンピテンシー選択

No.1〜135 の中から、該当するコンピテンシーを原文のまま選択させる。

STEP0
期限・目標文入力

期限と、ユーザー自身の言葉による目標文を入力する。

STEP1
行動計画の設計

達成のための行動を必ず3つ設定する。具体的で、今日から実行可能であること。

STEP2
最終確認

選択項目、目標文、行動①〜③を確認し、提出前レビューを行う。

STEP3
レポート出力

Word提出前提の構成でレポート生成し、固定締め文を末尾に入れる。

重要制約として、選択済みのステージ名・グレード名・カテゴリ名・コンピテンシー原文には一切手を加えず、そのまま表示・出力すること。
3

マスターデータ要件

マスター内容備考
qualitative_stagesPDF準拠のステージ一覧表示順保持必須
qualitative_gradesPDF準拠のグレード一覧ステージ紐付けあり
qualitative_categories定性目標カテゴリ一覧言い換え禁止
competenciesNo.1〜135、原文全文No / 本文 / 並び順 / 有効フラグを保持
4

画面要件

入力画面

  • 1問ずつ表示
  • 選択肢はPDF原文そのまま
  • 現在ステップ表示
  • 途中保存・再開
  • 右側に確定済みサマリー

確認画面

  • 目標・選択項目・行動3つを一覧表示
  • 編集戻り導線
  • 提出前プレビュー
  • Word/PDF出力ボタン
5

データ要件

主テーブル案

  • qualitative_goal_sessions
  • qualitative_goal_answers
  • qualitative_goal_actions
  • qualitative_goal_reports

保存項目

  • name / department
  • deadline / goal_text
  • stage_id / grade_id / category_id / competency_id
  • action_1 / action_2 / action_3
  • status / submitted_at / approved_at
6

AI利用ルール

分類可否内容
質問進行可1問ずつ進行、入力不足時の再質問、行動の具体化促進
選択項目の変換不可ステージ・グレード・カテゴリ・コンピテンシーの要約や言い換えは禁止
評価・採点不可正解、模範解答、採点、比較は禁止
文書整形限定可Wordレポートの体裁整形のみ可。原文改変は禁止
7

出力要件

Wordレポート構成

  1. 表紙情報：名前 / 部署 / 作成日
  2. 目標：期限 / 目標文
  3. 選択した定性目標：ステージ / グレード / カテゴリ / コンピテンシーNo / 原文全文
  4. 行動計画：行動① / 行動② / 行動③

本文末尾には必ず次の固定文言を入れる：「この目標は、今日決めたこの一歩から始まります。」

8

非機能要件

  • PDF原文の整合性を保つこと
  • スマートフォンでも操作可能であること
  • 途中保存・再開・差戻し履歴を保持すること
  • 人事評価管理画面から一覧・提出状況確認ができること
  • 操作ログ・変更履歴を取得できること
